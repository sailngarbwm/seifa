seifa
======================================================================

Welcome to the documentation for seifa, a Python interface to several Australian socio-economic indices.

The Australian Bureau of Statistics (ABS) publishes a variety of indexes for the Australian
economic environment. These include a variety of indexes designed to measure socio-economic advantage. `ausdex` makes these data
available in a convenient Python package with a simple programatic and command line interfaces.
